# Credit Card Validator (C Project)

## Description
This program validates credit card numbers using the Luhn algorithm and identifies the card type (VISA, MasterCard, AMEX).

## How it works
- Takes a credit card number as input
- Applies Luhn algorithm to verify validity
- Determines card type based on number patterns and length

## Card Types Supported
- AMEX (15 digits, starts with 34 or 37)
- MasterCard (16 digits, starts with 51–55)
- VISA (13 or 16 digits, starts with 4)
